## 👤 Sobre Mim

Atualmente, estou focado em construir uma base sólida em desenvolvimento de software, algoritmos e engenharia de dados. Utilizo este espaço para centralizar meus estudos universitários, projetos práticos e iniciativas pessoais, buscando sempre aplicar as melhores práticas de mercado e versionamento de código.
